# 自己PR案：ベンチプレス100kgへの挑戦と研究への応用

## アピールポイント
「粘り強さ」と「試行錯誤による継続的な改善力」。
ベンチプレスのフォーム改善プロセス（YouTube研究、実践、修正）を、研究開発（アルゴリズム選定、実装、検証）になぞらえて説明します。

---

### パターンA：問題解決力・泥臭さ重視（おすすめ）
**[...大学2年生の夏からジムに通い始め〜昨年の夏に目標だった100kgを達成しました。]**

この「地道な試行錯誤を継続する力」は、現在の姿勢推定の研究においても活かされています。
私の研究テーマは、単眼カメラのみを用いたバーベルの挙上速度計測システムの構築です。開発当初は、カメラのノイズにより計測値が安定せず、実用的な精度が出ないという壁に直面しました。
そこで私は、ジムでのフォーム改善と同様に、何が根本的な原因なのかを徹底的に分析しました。既存のライブラリ（YOLOやMediaPipe）をただ使うだけでなく、それぞれの長所と短所を論文やGitHubで調査し、数百回にわたるテスト動作のログを解析しました。その結果、遠距離からの撮影では認識精度が落ちるという課題に対し、「YOLOで位置を特定してからMediaPipeで詳細を見る」というハイブリッド手法を考案し、実装に至りました。
このように、目標達成のために現状を分析し、粘り強く改善を繰り返す姿勢は、貴社における困難な開発課題に対しても必ず貢献できると確信しています。

### パターンB：「好き」を「技術」に昇華させる力（情熱重視）
**[...大学2年生の夏からジムに通い始め〜昨年の夏に目標だった100kgを達成しました。]**

この「地道な試行錯誤を継続する力」は、現在の姿勢推定の研究においても活かされています。
自身がトレーニングに打ち込む中で感じた「高価な計測機器がないと最適な負荷設定ができない」という課題を技術で解決したいと考え、画像処理を用いた自動計測アプリの開発に取り組みました。
開発においては、単に動くものを作るだけでなく、実際に自分がジムで使いやすいものにすることにこだわりました。「縦画面で撮りたい」「友達に撮ってもらう時は距離が遠くなる」といった現場のリアルな制約条件を一つ一つクリアするために、グリップ幅を利用した独自の補正ロジックなどを考案・実装しました。
自分の情熱を原動力に、ユーザー目線で粘り強く製品をブラッシュアップしていく姿勢で、貴社のサービス開発に貢献したいと考えています。

### パターンC：技術的な探究心重視
**[...大学2年生の夏からジムに通い始め〜昨年の夏に目標だった100kgを達成しました。]**

この「地道な試行錯誤を継続する力」は、現在の姿勢推定の研究においても活かされています。
研究では、単眼カメラの映像から3次元的な挙上速度を推定するという課題に対し、精度の向上が難航しました。しかし、ベンチプレスで培った「フォームの違和感を放置しない」という感覚と同様に、データのわずかな異常値も見逃さずに原因を究明し続けました。
具体的には、手指の検出ズレが速度のノイズになっていることを突き止め、平滑化フィルタのパラメータを0.01単位で調整したり、先行研究のハイブリッド構成を取り入れるなどの改修を行いました。その結果、最終的には市販の専用デバイスと同等の精度（誤差10%以内）を実現することができました。
困難な技術課題に対しても、妥協せず細部までこだわり抜く粘り強さを発揮し、成果を出していきたいと思います。
